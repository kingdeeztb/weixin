# 关于项目
此项目是作者开发的并成功上线的微信小程序源码.
代码整合优秀的开源资源,进行二次开发.
可以作为一款优秀的"微信天气小程序"入门教程.研究微信小程序编程思想.
有需要请联系作者:kingdeeztb#163.com
# 使用方法
1.下载 "微信开发者工具"-->项目-->导入新项目即可使用.
